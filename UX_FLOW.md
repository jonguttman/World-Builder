# UX Flow (MVP)

## Primary Flow
Launch -> Level Select -> Level Brief -> Planet View -> Recap -> (Codex Updates) -> Back to Level Select

## Level Select
- 1–3 free as "Training"
- 4–10 locked with Pack 1 badge
- 11–20 locked with Pack 2 badge
- Tap locked: opens Store Preview

## Planet View
- Top: Time controls (pause/play, speed)
- Right: Overlay selector (Temp, Nutrients, Pop, Ice)
- Bottom: Intervention tray
- Tap tile: inspector card
- Long press: forecast

## Recap Screen
- Timeline playback, new discoveries, world score
- After Level 3: Paywall preview + purchase CTA
