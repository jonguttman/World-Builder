## Campaign structure
Each level is a handcrafted scenario with:
- Starting planet parameters (physics/atmo/hydro + geography seed)
- Allowed interventions + total energy budget
- Objective conditions + success duration
- Failure conditions + fail states
- Time controls (caps or warnings per level)

## Codex (progression)
Progression is knowledge, not XP.

Codex categories:
- Species, Body Plans, Biomes, Planetary Systems, Evolutionary Events, Failure Modes, Rare Phenomena, Historic Worlds

Codex entry structure:
- Name / Category
- Discovery source (level + world seed)
- Conditions required
- Simulation facts (thresholds, ranges)
- Flavor text (short)
- Visual record (silhouette + biome card)
- Cross references

Unlock triggers:
- Trait stabilized (duration threshold)
- Speciation event observed
- Major event occurs
- Failure mode repeated
- Rare condition sustained

UI:
- Codex index
- Category browser
- Entry viewer
- Filters + bookmarks
- "New discoveries" feed after each run/level
